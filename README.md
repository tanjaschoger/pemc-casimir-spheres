# pemc-casimir-spheres

`pemc-casimir-spheres` is a Python code to compute the Casimir interaction for **bi-isotropic objects**, with a focus on **PEMC (Perfect Electromagnetic Conductor) spheres**. 

## Features

- Compute Casimir forces and energies between **two spheres** or a **sphere and a plane**.  
- Supports **PEMC boundary conditions** and general bi-isotropic materials.  

## Documentation

Detailed documentation for the classes and functions in this code can be built using **Sphinx**.
First, navigate to the `docs` folder

```bash
cd docs
```

Then build the documentation with

```bash
make html
```

The HTML files can be found in 

```bash
docs/build/html
```
