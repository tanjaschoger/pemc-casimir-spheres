# pemc-casimir-spheres

`pemc-casimir-spheres` is a Python code to compute the Casimir interaction for **bi-isotropic bodies**, with a focus on **PEMC (Perfect Electromagnetic Conductor) spheres**. It supports both **sphere-sphere** and **plane-sphere** geometries.

## Features

- Compute Casimir forces and energies between **two spheres** or a **sphere and a plane**.  
- Supports **PEMC boundary conditions** and general bi-isotropic materials.  
- Simple Python script suitable for research and scientific applications.
